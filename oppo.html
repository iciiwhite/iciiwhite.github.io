01. Driver Installation

Windows strictly blocks unsigned drivers. You must disable this first, or the phone won't connect.

Action: Disable Driver Signature Enforcement
  1. Restart PC holding Shift key.
  2. Go to Troubleshoot > Advanced Options > Startup Settings > Restart.
  3. Press 7 or F7 when menu appears.
Action: Install QDLoader

Run the Qualcomm Driver setup (usually Setup.exe). Select "WWAN-DHCP" if asked (rare), usually just "Next" until finished.

02. Prepare the Payload

  1. Extract the Firmware ZIP file to a folder (e.g., Desktop/A57_Rescue).
  2. Open the folder. You should see MsmDownloadTool.exe.
  3. Right-click the tool and select Run as Administrator.
  4. The tool will launch. It will show a list of ports (likely blank).
  5. In the "Select Model" dropdown (if present), choose A57 or CPH1701.

03. The "Black Screen" Handshake

We need to enter EDL (Emergency Download Mode). The screen will stay OFF. This is normal.

1

2

Hold Volume UP and Volume DOWN simultaneously.

3

While holding buttons, plug the USB cable into the PC.

4

Listen for the Windows "Device Connected" sound. Release buttons.

Device Manager Check:

Ports (COM & LPT)

Qualcomm HS-USB QDLoader 9008 (COM3)

Check Point: If you see the Oppo Logo or battery charging icon, you failed. Unplug, power off, and try again. The screen MUST be black.

  2. You should see one of the ports listed as "Connected".
